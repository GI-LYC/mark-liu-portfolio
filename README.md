# 刘奕辰 Mark / Portfolio

电影感暗色汽车视觉设计作品集网站，内容取自当前目录中的个人简历、作品图片与视频素材。

## 本地运行

需要 Node.js 18 或更新版本。

```powershell
npm run dev
```

打开 `http://127.0.0.1:4173`。

项目不依赖第三方 npm 包，本地服务器支持视频分段请求，因此详情页中的 MP4 / M4V 作品可以直接拖动播放。

## 页面

- `index.html`：首页 Hero、代表作品与能力介绍
- `portfolio.html`：筛选式卡片作品集
- `project.html?id=audi-earth-day`：项目详情、大图查看与视频播放
- `about.html`：个人简介、技能与服务方向
- `contact.html`：联系信息
